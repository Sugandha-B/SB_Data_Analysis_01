# Zomato Restaurant Data Analysis

An Exploratory Data Analysis (EDA) project on the Zomato Restaurant Dataset using Python. This project focuses on understanding restaurant trends, customer preferences, pricing patterns, and factors influencing restaurant ratings through data cleaning, visualization, and statistical analysis.

## Project Overview

This project applies data preprocessing and exploratory data analysis techniques to a Kaggle Zomato dataset. The objective is to extract meaningful business insights that can help understand customer behavior and restaurant performance.

##  Objectives

- Clean and preprocess raw restaurant data
- Analyze restaurant ratings and customer votes
- Explore pricing patterns across restaurants
- Identify popular cuisines and locations
- Study the impact of online ordering on ratings
- Visualize relationships between different variables
- Generate business insights from the data

##  Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

##  Dataset

- Source: Kaggle
- Dataset: Zomato Restaurant Dataset

> *The dataset is publicly available on Kaggle and is used for educational purposes.*

##  Data Preprocessing

- Removed missing and duplicate values
- Handled inconsistent data formats
- Renamed columns where necessary
- Converted data types
- Prepared data for visualization and analysis

## Visualizations

The analysis includes:

- Histograms
- Bar Charts
- Count Plots
- Scatter Plots
- Heatmap
- Word Cloud

##  Key Insights

Some of the insights obtained include:

- Most restaurants fall within a mid-range rating.
- Certain cuisines dominate the restaurant market.
- Higher customer votes generally correspond to better ratings.
- Online ordering is more common among highly rated restaurants in several locations.
- Restaurant pricing varies significantly across locations.
